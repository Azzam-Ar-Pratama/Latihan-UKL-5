# Latihan-UKL-5
Tujuan
Program ini menghitung rata‑rata nilai dari sejumlah siswa yang dimasukkan oleh pengguna.

Input yang diminta:
Jumlah siswa(integer).
Nilai tiap siswa (angka).

Cara kerja:
1. Baca jumlah siswa. Jika bukan integer atau ≤ 0, program memberi pesan dan berhenti.
Untuk setiap siswa (loop dari 1 sampai n):
2. Minta input nilai.
Jika input bukan angka, minta ulang sampai valid.
Jika nilai negatif, tampilkan pesan dan ulangi input untuk siswa itu.
3. Tambahkan nilai valid ke penjumlahan (sum).
Setelah semua nilai masuk, hitung rata‑rata = sum / n.
4. Cetak rata‑rata dengan 2 angka di belakang koma.
5. Tutup Scanner.

Output:
Menampilkan teks: "Rata-rata nilai dari X siswa: Y.YY"
